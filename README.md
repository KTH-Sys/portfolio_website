## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Glassmorphism effects with gradient accents (orange to purple)
- **Smooth Animations**: Fade-in effects and smooth scrolling navigation
- **Interactive Navigation**: Fixed navbar with smooth scroll to sections
- **Background Image**: Custom neural network-themed background with overlay
- **Project Links**: Direct links to GitHub repositories and Devpost projects

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with glassmorphism and gradient effects
- **Fonts**: Inter (Google Fonts)
- **Deployment**: Static site (can be hosted on GitHub Pages, Netlify, Vercel, etc.)

## Local Development

1. **Clone or download this repository**

2. **Open the website**:
   - Simply open `index.html` in your web browser, or
   - Use a local server:
     ```bash
     # Using Python 3
     python3 -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     ```
   - Then navigate to `http://localhost:8000` in your browser

## File Structure

```
portfolio/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # JavaScript for interactivity
├── background.png  # Background image
└── README.md       # This file
```

## Customization

- **Colors**: Modify CSS variables in `:root` section of `styles.css`
- **Content**: Update directly in `index.html`
- **Background**: Replace `background.png` with your own image
- **Styling**: Adjust opacity and overlay in `styles.css`

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
