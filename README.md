# Portfolio Website

Personal portfolio website for Kyaw Thi (Eric) Ha.

## Live Sections

- **Hero** - Background image with typing animation
- **About** - Bio and contact details
- **Skills** - Languages, frameworks, tools, and libraries
- **Resume** - Education and experience
- **Projects** - Filterable project cards with GitHub links
- **Contact** - Contact info and form

## Tech Stack

- HTML5, CSS3, JavaScript
- Fira Code (monospace font)
- Bootstrap Icons
- Responsive design

## Setup

Open `index.html` in a browser or deploy to any static hosting (GitHub Pages, Netlify, Vercel).
